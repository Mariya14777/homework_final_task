Основные команды Git

  • git init - создать новый репозиторий
  • git cione- клонировать удалённый репозиторий
  • git status-показать текущий статус файлов
  • git add. - добавить все изменённые файлы
  • git push - отправить локальные коммиты на сервер
  • git pull - получение и отправка изменений с сервера
  • git log - показать историю коммитов
